Сделать запускаемую программу:
$ python3 sky.py [options] +
Команды:
	1) листинг файлов в облаке, в указанном каталоге (только имена) +
	2) подробный листинг файлов в облаке, в указанном каталоге (имена + метаданные) +
	3) просмотр списка изменившихся локальных файлов +
	4) просмотр списка файлов, требующих синхронизации +-
	5) синхронизация указанных файлов в облако -
	6) синхронизация указанных файлов из облака -
Выделить всю работу с облаком в отдельный класс

Состояние каталога сохранять в формате json, в файле .cloud, расположенном в синхронизруемом каталоге
Если файл .cloud создать нельзя, программа не должна ломаться

Работающее отслеживание изменений в локальном каталоге +
Загрузка файлов из облака локально +
Логин принимается через параметры запуска, пароль — через getpass +
